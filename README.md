# Agentic CDO

A collection of Claude Code skills, agents, and plugins for city Chief Data Officers.

Each module is a reusable tool for a specific CDO workflow — performance management, open data, AI governance, and more.

## Structure

```
01-open-data/                   # Open data publishing and quality workflows
02-performance-management/      # AI-enabled metrics and performance frameworks
03-data-management/             # Data inventory, readiness, and governance tools
```

## Modules

| Module | Type | Description |
|--------|------|-------------|
| `02-performance-management/ai-enabled-metrics` | Skill | Brainstorm AI-enabled performance measures using a four-category typology |
| `03-data-management/data-for-ai-assessment` | Skill | Assess open government datasets for AI-readiness using a synthesized rubric |

## Usage

These modules are designed for use with [Claude Code](https://claude.ai/code). Install a skill or plugin via the Claude Code plugin system and invoke it by name or trigger phrase — each module's `SKILL.md` documents its invocation patterns.

## Audience

City CDOs, performance managers, data leads, and innovation teams in local government.
